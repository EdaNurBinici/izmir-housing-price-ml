# Documentation

This directory contains project documentation and visual assets.

## Structure

- `images/` - Report graphics and visualizations generated during EDA

## Report Graphics

The following visualizations are generated by `grafik.py`:

1. **Rapor_Grafik_1_FiyatDagilimi.png** - Price distribution histogram
2. **Rapor_Grafik_2_M2_Fiyat.png** - Area vs Price scatter plot
3. **Rapor_Grafik_3_Ilceler.png** - District-based price analysis
4. **Rapor_Grafik_4_LogFiyat.png** - Log-transformed price distribution
5. **Rapor_Grafik_5_Boxplot.png** - Price boxplot by district
6. **Rapor_Grafik_6_M2_Degeri.png** - Price per square meter by district
7. **Rapor_Grafik_7_Korelasyon.png** - Feature correlation heatmap

## Generating Graphics

To regenerate all visualizations:

```bash
python grafik.py
```

Graphics will be saved to `docs/images/` directory.
